# MemoryCards

Simple Memory Cards game created in Android Studio using Kotlin and Jetpack Compose.

---

## About the project

The application is a simple memory game where the player has to find matching pairs of cards.

Features:
- card matching logic
- move counter
- random card positions
- win detection
- game reset button

---

## Technologies

- Kotlin
- Android Studio
- Jetpack Compose
- Android Emulator
